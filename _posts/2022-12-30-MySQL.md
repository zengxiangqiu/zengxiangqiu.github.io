##  Command

```bash
# connect
mysql --user=root --password nshop -h 127.0.0.1
# backup
mysqldump -u root -p nshop -h 127.0.0.1 --column-statistics=0 --create-options -v > nshop_20230220.sql
```

## 开启 log_bin

> mysql-8.0.33 配置文件path`/etc/my.cnf`

```ini
[mysqld]
log_bin = /var/log/mysql-bin.log
```
