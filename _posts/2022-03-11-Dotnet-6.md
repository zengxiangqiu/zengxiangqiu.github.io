.NET Core 编码支持
默认情况下，.NET Core 不提供除代码页 28591 以外的其他任何代码页编码和 Unicode 编码，例如 UTF-8 和 UTF-16。

```csharp
nuget install package System.Text.Encoding.CodePages

//注册编码表
Encoding.RegisterProvider(CodePagesEncodingProvider.Instance);
```
